# Web UI 与 CLI 功能对齐计划

## 目标
把 CLI 版本刚才优化的功能同步到 Web UI

---

## CLI 版本优化清单（需要同步）
1. ✅ 资源库删除功能（带二次确认和备份）
2. ✅ API 协议选择
3. ✅ 自动发现模型
4. ✅ 手动管理模型
5. ✅ 搜索服务增强（官方+第三方，3个官方搜索服务）
6. ✅ 大小写不敏感（后端 API 处理）
7. ✅ 所有操作自动备份
